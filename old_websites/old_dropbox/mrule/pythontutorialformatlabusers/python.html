Learn Python, for Matlab Users

Python is a free, open source scripting language that runs on all major operating systems (wiki, docs, tutorial).
The numerical library Numpy provides fast linear algebra in Python (docs, wiki, examples).
The plotting library Matplotlib provides graphical tools for data visualization (wiki, examples, gallery).
The library Scipy is built on top of Numpy and Matplotlib, and provides most Matlab features, and many more (docs, wiki, get started).

The document Numpy for Matlab users details the major differences between Scipy and Matlab. I find this cheat-sheet helpful when switching between Matlab and Scipy.

Why?

  • Scipy is free.
    • Workflow will not be interrupted by license server outages
    • People outside of universities will be able to use your code
  • The Python language
    • Support for general purpose programming
    • Widely regarded as more readable and easy to learn
  • Scipy is awesome
    • More advanced plotting tools generate more professional looking plots
    • Open source design means that the library is evolving and rapidly responsive to user needs
  • Everyone is doing it!
    • You may need to be able to use, read, understand, and write Python code in order to collaborate
    • You may find learning a new language entertaining
    • Fluency in Python opens doors to solving many other types of problems with programming

Why not?

  • If you have no experience with at least one programming language, learning Python will take a bit of time
  • If you do not plan to share or collaborate on code, then the time investment to learn Python may not be worth it
  • Matlab is a good domain-specific language for linear algebra: its syntax for matrix manipulation can be more compact
  • If you are uncomfortable with command line interfaces, you may have difficulties.

Warnings

Python array indexies start at 0, like in C, Java, and most other programming languages. Matlab array indexes start at 1, which is a hold-over from Matlab's origins in Fortran, and is relatively rare amongst programming languages.

Matlab is a domain specific language designed specifically for linear algebra. Python is a general purpose programming language with support for Matlab-like functionality through the Scipy library. Do not expect to be able to use straight python for efficient numerical computations -- you should make a habit of using the Numpy array datatype.

Python has many datatypes, Matlab has only a handful. This can cause some confusion, especially when variables are not the datatype that you expect.
