Github repositories, user michaelerule

Neuroscience

GLM moment closure.

Example code for Rule and Sanguinetti 2018, which explores an approach for analyzing autoregressive point-process models (used to capture neuronal spike-train statistics) as state-space models.

Neurotools: computational neuroscience python routines.

A compiliation of various python functions that I use routinely in computational and statistical neuroscience.

neuralfield.

This project builds on the WebGL examples for GPGPU computing to develop in-browser WebGL implementations of Wilson-Cowan style neural field equations. The project website examples can be browsed here.

CGID waves example.

Example code and datasets corresponding for Phase reorganization leads to transient Î²-LFP spatial wave patterns in motor cortex during steady-state movement preparation. Related code is also stored in this repository, although it is in a non-maintained non-working state.

Hobbies

Mandalas.

A collection of my manala artwork baseed on quasicrystals (aperiodic crystals).

Uno9341TFT.

Optimized graphics drivers for the 9341TFT touch-screen displays on the Arduino Uno, as well as a 3D rendering engine for this platform. (Hackaday post)

webgpgpu.

Tutorial examples for accelerating 2D rendering in the browser using WebGL, building up to neural-field simulation demos.

Video Feedback.

Various routines for rendering video-feedback fractals. This includes several incomplete and non-maintained projects. The repository contents can be browsed here.

LED matrix.

Files related to hobby LED matrix display projects.

tinymarquee.

Source code and design files for AtTiny24 based scrolling marquee (Hackaday post).

quasicrystal.

Small Java code examples and outputs for rendering quasicrystal patterns.
